#第一种
```python
a = 0          #先定义变量a，并赋值
while a < 5:   # 设定一个放行条件：a要小于5，才能办事
    a = a + 1  # 满足条件时，就办事：将a+1
    print(a)   # 继续办事：将a+1的结果打印出来
```
