湖南长沙居民自建房倒塌事故已有10人获救5人遇难

中国新闻网

2022-05-05 00:39 中国新闻网官方帐号

中新网长沙5月5日电 (记者 刘双双 杨华峰)记者5日从湖南长沙居民自建房倒塌事故新闻发布会上获悉，截至目前已有10人获救，5人遇难。

5月5日凌晨，长沙居民自建房倒塌事故第五次新闻发布会举行，通报事故救援最新情况。

会上，长沙市委副书记、市长、长沙居民自建房倒塌事故应急救援和善后处置指挥部指挥长郑建新介绍，3日下午以来，倒塌房屋东侧楼房的西墙险情加重，指挥部根据现场救援进展情况，组织专业队伍，通过雷达、全站仪等多种先进设备，从不同方位对倒塌房屋两侧建筑开展不间断的安全监测，同时采取加固措施，为工程机械作业创造条件，避免发生次生灾害。
