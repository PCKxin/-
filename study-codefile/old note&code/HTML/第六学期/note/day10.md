## 表格

```html
<table>   //表格开始
    <tr>   //行开始
        <td>内容</td> //列， 单元格
        <th>内容</th>  //同上 ，字体默认加粗了 
    </tr>


</table>

border="1px"       //边框厚度
width="100px"      //宽度
height="100px"     //高度
bgcolor="pink"    //背景颜色
background="图片地址"   //加图片











```
