实现方法：
找出对应类名
row 开启弹性布局 子元素换行
col-数字 1-12 代表占用的列数
col-断点-数字
col-断点(不加数字) xs sm md lg xl
当小于断点时, 独占一行
当大于等于断点时, 会有响应式的变化
xs 小于576px
sm 576 540
md 768 720
lg 992 960
xl 1200 1140
xxl 1400 1320
栅格体系中的嵌套
正常情况: 一行分为12列
每一列可以用col/col-数字
每一列再分为12列

g表示列间距
rem 单位可以和px相互转换
1rem 默认情况下 = 16px
浏览器默认的font-size的值有关系
可以手动修改rem和px的比例关系
只需要在html标签中设置font-size的值
1rem = 16px
g-0 0rem 0
g-1 0.25rem 4px
g-2 0.5rem 8px
g-3 1rem 16px
g-4 1.5rem 24px
g-5 3rem 48px
gx 指的是左右的列间距
gy 指的是上下的列间距
g 指的是上下左右的列间距

t = top
b = bottom
s = left
e = right
外边距: m
内边距: p
mx-auto = margin: 0 auto
me-auto = margin-left: auto
start end 代表的是文本的方向

d-flex 开启弹性布局
flex-row/column 定义主轴的方向
justify-content-start/end/center/between/around/evenly
定义主轴的对齐方式
