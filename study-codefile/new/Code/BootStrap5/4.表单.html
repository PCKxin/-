form-control 表单的类名
form-control-lg 表单控件大
form-control-sm 表单控件小
form-text 表单的说明文字
form-control-plaintext 表单控件纯文本
form-control-color 表单控件颜色
form-control-range 表单控件范围
form-check-input 表单检查输入
form-check-label 表单检查标签
form-check-inline 表单检查内联
form-label 表单标签
form-select 表单选择
form-select-lg 表单选择大
form-select-sm 表单选择小
form-select-multiple 表单选择多个
form-select-disabled 表单选择禁用
form-select-plaintext 表单选择纯文本
form-select-sm-plaintext 表单选择小纯文本
form-select-lg-plaintext 表单选择大纯文本
form-select-sm-multiple 表单选择小多个
form-select-lg-multiple 表单选择大多个
~
form-switch 表单开关
multiple 多个
aria-describedby : 设置表单组的id
aria-label : 设置表单组的标签
form-floating : 聚焦时动画
is-invalid : 设置表单控件的状态为无效
input-group : 表单组
input-group-text : 表单组的文本
input-group-addon : 表单组的附加项
