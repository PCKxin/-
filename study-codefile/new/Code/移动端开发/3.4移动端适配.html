em适配

1em = 16px
em大小根据自身的字体大小来计算
如果没有设置字体大小, 则会一直向上冒泡, 直到body, 都没有就是默认16px

rem适配

rem 是根据根元素(节点)的字体大小来计算的
默认1rem = 16px
如需设置需要在html中设置font-size
html{ font-size: 100px; }
1rem = 100px
针对不同尺寸设备的适配等比例显示, 需要计算不同的rem值
比如设计稿尺寸是 640 * 1080 font-size是100px
这个设计稿的rem值就是 6.4rem * 10.8rem
如果想在宽 X 的屏幕上使用
计算公式就是 X / 640 * 100

vw适配

vw 是根据视口宽度来计算的
1vw = 1%的视口宽度
1vh = 1%的视口高度
1vmin = 1vw和1vh中较小的那个
1vmax = 1vw和1vh中较大的那个

pt适配

pt 是根据屏幕分辨率来计算的
1pt = 1/72英寸
1英寸 = 2.54cm
1cm = 10mm
1pt = 1/72 * 2.54 * 10 * 10
1pt = 0.3528mm
1mm = 2.8346pt
1px = 0.75pt
1pt = 1.3333px
