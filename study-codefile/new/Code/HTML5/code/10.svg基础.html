canvas 和 svg 不同?

canvas 是位图 放大 会失帧
canvas是一个标签 只是一个容器
getContext('2d')返回的对象来绘制
应用领域 范围比较广
svg 是矢量图 放大 不会失帧
基于xml(关于xml特点使用规范6点)
使用标签绘制
应用领域 简单的图标绘制 地图
最主要的缺点 对颜色的支持比较少 没有canvas支持的多

svg引入四种方式

1.使用img标签或者background-image引入
2.使用object标签引入 用法:
object data="path" type="image/svg+xml" /object
3.使用iframe标签引入 用法:
iframe src="path"
4.直接使用svg标签

circle 圆形
rect 矩形
line 直线
fill填充颜色 stroke边框颜色 默认是有填充颜色（黑色），边框是没有的
fill和stroke有一个特殊的值 none/transparent表示透明
这几个属性是公有属性 fill stroke stroke-width stroke-opacity

g标签

group g表示分组的意思, g标签可以理解为将里面所有的子标签当作一个整体
g标签中定义的属性, 会被里面子标签继承 属性是有限制的 只有公有
属性才能被定义到g标签中, 可以给g标签加一个id或者class, 操作g标签
就等于操作了里面所有的子标签

icon案例

path标签
path标签是用来绘制路径的, 有一个d属性, 属性值:
M 大写的M 表示绘制第一个点 canvas.moveTo(x,y)效果一样
L 表示绘制其他的点 跟canvas.lineTo(x,y)效果一样
Z 表示闭合路径 和canvas.closePath()效果一样
