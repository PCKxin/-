第一种
oninvalid 属性
用于在验证表单输入字段时触发自定义操作
当用户尝试提交无效数据时, 浏览器会自动验证表单字段
通过使用 oninvalid 属性，我们可以在验证失败时显示自定义的错误消息或执行其他自定义操作
在数据发送到服务端之前, 需要对表单元素按照约束条件进行验证, 验证不通过, 会收到invalid通知
oninvalid 事件验证 valueMissing 要和 required 配合使用
