媒体引入

两种方式引入媒体文件
1、直接引入
2、使用source标签引入

标签属性解释 controls 显示或隐藏控制菜单, 属性值为controls时显示, 无属性值时隐藏
autoplay 自动播放
muted 静音
loop 循环播放
poster 视频播放前的预览图片, 如果加入autoplay属性的话, 这个属性会失效, 视频的大小会跟这张预览图片大小有关系
preload 预加载, 属性值为auto时, 页面加载时会加载视频, 属性值为none时, 页面加载时不会加载视频
---js---
paused属性 表示当前播放的状态是否暂停, 暂停返回true, 播放返回false
ended属性 true当前视频/音频播放结束, false未结束
onended事件 播放结束时会调用, 注意 如果加上loop属性, 这个事件永远不会执行
