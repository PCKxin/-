基本属性

animation-name: 自定义动画名称

animation-duration: 动画持续时间

animation-delay: 动画延迟时间

animation-iteration-count: 动画播放次数

animation-timing-function: 动画的速度曲线

linear 匀速

linear

ease 先快后慢

ease

ease-in 先慢后快

ease-in

ease-out 先快后慢

ease-out

ease-in-out 先慢后快再慢

ease-in-out

animation-direction 定义动画播放顺序

两个属性值
alternate
效果生效的前提是，动画播放的次数大于等于2
动画在奇数次播放时，按照正常的顺序播放
偶数次播放时，按照反方向播放
第1, 3, 5, 7....次播放正向播放
第2, 4, 6, 8....次播放逆向播放
正向播放的意思 从 0% ---> 100%
逆向播放的意思 从 100% ---> 0%

alternate-reverse
动画在奇数次播放时，按照反方向播放
偶数次播放时，按照正常的顺序播放

animation-fill-mode 控制动画播放前的状态

backwards , 在延迟的这段时间, 保持动画第一帧的状态
forwards , 动画播放完毕之后, 保留最后一帧的状态
both , 既保留延迟期间第一帧状态, 也保留动画结束之后最后一帧状态

animation-play-state 控制动画的播放状态

两个属性值
running , 动画正在播放
paused , 动画暂停播放
