什么是媒体查询

当屏幕尺寸发生改变时, 会根据设置的不同尺寸, 设置不同的css样式
例如: 当屏幕宽度为1200时, div为红色
当屏幕宽度为960时, div为黑色
媒体查询会自动帮我们监听屏幕(浏览器)的尺寸

媒体查询的特点

最大的特点: 一套代码可以兼容pc和移动端, 减少开发成本
注意: 使用媒体查询时, 要对布局的整体把握非常到位,可以画草图, 想好了再写

媒体查询的三种引入方式

在link标签中使用media属性
在style标签中使用@media screen and ( )
在style标签中使用@import url( ) screen and ( )
注意
@import要写在style表开头

书写顺序问题

如果写max-width:属性值 这个属性值要从大到小去写
如果写min-width:属性值 这个属性值要从小到大去写
