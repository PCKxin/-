定义在父元素上的属性：

1.开启弹性布局
2.定义主轴的方向
3.主轴的排列方式
4.交叉轴的排列方式（单行，多行）
5.是否换行

定义在子元素上的属性：

1.flex
2.flex-grow
3.order
4.align-self

flex

flex属性 flex:num

flex属性是把父元素剩余空间进行平分

子元素占父元素的几分之几

如果在子元素中定义flex属性

当主轴为水平方向时 width 会失效

当主轴为垂直方向时 height 会失效

flex-grow

flex-grow属性会保留原有尺寸

最后的计算结果为

原有尺寸 + (父容器 - 所有子元素占有的空间) x 子元素占父元素的几分之几

order属性值默认为0 , 值越小越靠前 , 支持负数
