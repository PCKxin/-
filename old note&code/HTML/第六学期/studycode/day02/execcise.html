第一，学会微笑

微笑，一个多么简单的动作，只需要将嘴角轻轻上扬，就能展示一个开心的表情。
没有人会为你的痛苦买单，心情是自己的，时刻提醒自己：笑一笑，没什么大不了！
那些爱笑的人并非没有痛苦，只不过他们善于把不快乐推一推，好好享受当下。 无论如何，开心也是一天，不开心也是一天。
好日子是自己过出来的，你的笑容才是最美的风景。

第二，学会自我欣赏

你的成绩不够优秀，你长得不够好看，你工作的效率总是比不上其他人……但你有你自己的优点，你有洁白如雪的品行、乐观向上的人生态度。
同学喜欢和你交往，因为你和善；朋友觉得你让人舒服，因为人好；你工资平平却足够生活，有自己的空余时间去做很多喜欢的事情。
每个人都如同被咬过的苹果，没有完美无缺的。即使你只是一个普普通通的人，你也可以是一个充满爱心、充满自信的普通人。

第三，学会自己照顾自己

没有人会搀扶你一辈子，能和你相处最久的人就是你自己，你最该依赖的人也是自己。
人要学会自己照顾自己，自己激励自己。凡事少依赖其他人一些，多依靠自己一些。
靠自己，什么都不会倒。

第四，学会对痛苦说拜拜

爱情使人忘记时间，时间也会使人忘记爱情，不要让太多的昨天占据你的今天。
这一生，我们会遇到太多痛苦和无奈：痛苦爱而不得，无奈不想失去的人终会离去；痛苦总也无法过上想要的生活，无奈摆脱不了眼前的困顿。
但有时候你要站远一点，才能看得更透彻。你要学会和现下的痛苦打个照面，认认真真去面对，然后用你的方式去和它说再见。

第五，学会淡看得失

不要刻意去追求不属于你的，不要太过计较过去。没有路可以重走，如果有缘，错过了还会重来；
如果无缘，相遇了也会离开。 生活总不完美，有辛酸的泪，有失足的悔，有幽深的怨，有抱憾的恨。
学会淡视那些不完美，放大那些可能的完美，人生才会有新的希望。

第六，学会适当的“狠心”

对爱人，要“狠心”一点，让他承担责任、不忘进取，这样家庭才会更幸福。
对孩子，要“狠心”一点，该放手的时候放手，该独立的时候独立，这样才能让他们学会成长。
对自己，也要“狠心”一点，给自己一些适当的压力，逼自己更优秀一些。

第七，学会宽容

凡事太过较真，烦恼无处不在。
宽容自己，宽容他人，心平气和地工作生活。
选择宽容，不是怯懦，宽容了他人，也宽待了自己。
但宽容也有限度。有些不能原谅的错误，你可以不计较，但不能不长教训，屡错不改。

第八，学会坚强

没有遇到挫折，永远不会懂得自己的力量有多大。
你觉得自己遇到了黑暗，就再熬一熬，坚强一点，黎明总在前方，你要撑到那个时候。
如果你自己都放弃了，谁又能替你坚强呢？ 最后，学会珍惜。 对自己，珍惜你的健康，珍惜你的拥有。
同时，珍惜那些在乎你关心你的人，不要觉得他们的付出理所应当。 让我们感恩遇见，珍惜拥有！
